# Mr. Coder - Portfolio & Projects

## Overview
This is the portfolio website for Mr. Coder, a Shopify & WordPress Developer. The site showcases skills, experience, and a collection of high-performance ecommerce and content projects.

## Projects Page
The new `/projects` page is a dedicated section to browse all projects, categorized by platform:
- **Shopify**: Custom themes, apps, and conversion-optimized stores.
- **WordPress**: Headless builds, custom plugins, and editorial sites.

### Features
- **Grid Layout**: Projects are displayed in a responsive grid.
- **Platform Tabs**: Easily switch between Shopify and WordPress projects.
- **Project Cards**: Each card displays:
    - Website Image
    - Project Title
    - Tags (Tech stack/Goal)
- **Simple & Fast**: Designed for performance with minimal, tasteful animations.

## Tech Stack
- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion (used sparingly for performance)
- **Icons**: React Icons

## Getting Started

1.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```

2.  **Run the development server:**
    ```bash
    npm run dev
    # or
    yarn dev
    ```

3.  **Open the projects page:**
    Navigate to [http://localhost:3000/projects](http://localhost:3000/projects) to see the new projects section.
